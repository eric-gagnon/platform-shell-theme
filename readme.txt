== Description ==

Thème WordPress permettant de créer une plateforme de médialab collaborative.

* Voir aussi :
    * Le fichier README.MD dans le dossier de l'extension ou en ligne sur [Github](https://github.com/medialab-banq/platform-shell-theme/).
    * Le [wiki](https://github.com/medialab-banq/platform-shell-plugin/wiki) du projet.

== Installation ==

* Voir la procédure d’installation de la plateforme dans le [wiki](https://github.com/medialab-banq/platform-shell-plugin/wiki) du projet.
* Installer le thème comme n'importe laquelle autre thème WordPress.
